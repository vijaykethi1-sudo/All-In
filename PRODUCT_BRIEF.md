# Product Brief

## Product name
All-In Academy

## One-line pitch
A full-stack Texas Hold'em training lab where users practice live poker decisions and get instant coaching feedback.

## Core features
1. Playable heads-up Texas Hold'em simulator
2. Server-side hand-ranking evaluator
3. API-powered rule-based poker coach
4. Hand rankings and concept reference
5. Professional responsive dashboard UI

## Demo script for interviews
1. Open the dashboard and explain the product goal.
2. Start the simulator and walk through a hand.
3. Show how actions call the FastAPI backend.
4. Ask the coach, “What should I do?”
5. Open `/docs` to show the API contract.
6. Explain how the hand evaluator compares five-card combinations from seven available cards.
